常见sort算法总结

发表于2022-10-15|更新于2022-10-20
|阅读量:

Selection Sort

Process

  • Find the max/min value in the sequence
  • Put the selected value at the first place of the sequence
  • Repeat the process above

Code Implementation

Time Complexity

O(n2)\Omicron(n^2) O(n2)

Bubble Sort

process

  • Mean to make a sequence become an increment sequence

  • Compare two adjacent elements, put the bigger one at right

  • Repeat the exchange procedure until the biggest value in on the last

  • Repeat the bubble procedure above

Code Implementation

Time Complexity

O(n2)\Omicron (n^2) O(n2)

Rank Sort

Process

  • Produce a rank of the sequence
  • Sort the sequence by the rank

Code Implementation

Time Complexity

O(n2)\Omicron(n^2) O(n2)

Insertion Sort

Process

  • Insert x into the sorted sequence
  • Repeat the process above

Code Implementation

Time Complexity

O(n2)\Omicron(n^2) O(n2)

Radix Sort

Process

  • According to the last bit of the elements in the sequence, attribute them to the linked list
  • Arrange the elements on the base of the list
  • Target at the second last bit of the elements
  • Repeat until all the bits have been arranged

Code Implementation

quoted from baike.baidu.com

Time Complexity

O(d(n+radix))\Omicron(d(n+radix)) O(d(n+radix))
